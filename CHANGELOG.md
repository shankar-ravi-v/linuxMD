# Changelog

## v0.1.0-alpha

Initial experimental public release.

### Highlights

- System inventory
- Performance diagnostics
- Security assessment
- Deterministic health assessment
- Optional LLM-assisted interpretation
- OpenAI, Gemini, and DeepSeek provider support
- Structured JSON reports
- Provider-independent analysis pipeline
- Deterministic validation and fallback

### Known limitations

- Limited platform validation
- Limited hardware coverage
- Storage and network assessment depend on available telemetry
- Output schema may evolve
- Additional collectors planned
